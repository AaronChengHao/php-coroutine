PHP Coroutine
======

A lightweight library to implement coroutine by yield & Generator.

## Requirements

* PHP 5.5 or later

## Installation via Composer([packagist](https://packagist.org/packages/hhxsv5/php-coroutine))

```BASH
composer require "hhxsv5/php-coroutine:~1.0" -vvv
```
